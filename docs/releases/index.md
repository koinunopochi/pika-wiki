## リリースサイクル

pika-wikiは以下のリリースサイクルに従います：

- **機能リリース**: 月1回程度（新機能や大きな改善）
- **パッチリリース**: 必要に応じて（バグ修正やセキュリティ対応）
- **メジャーリリース**: 年2回程度（大規模な変更や破壊的変更を含む）

## 通知を受け取る

新しいリリースの通知を受け取るには：

1. GitHubリポジトリをWatchする
2. RSSフィードを購読する（実装予定）
3. お知らせAPIを定期的にチェックする

```bash
curl https://pika.lynxes.org/api/tools/announcements
```
